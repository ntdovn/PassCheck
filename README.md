# PassCheck

Ứng dụng kiểm tra độ mạnh mật khẩu, tạo mật khẩu ngẫu nhiên và kiểm tra dữ liệu bị rò rỉ.

## Tính năng

- **Kiểm tra mật khẩu**: Đánh giá độ mạnh của mật khẩu dựa trên các tiêu chí bảo mật
- **Tạo mật khẩu**: Tạo mật khẩu ngẫu nhiên an toàn với các tùy chọn tùy chỉnh
- **Kiểm tra rò rỉ**: Xác minh email hoặc mật khẩu có bị lộ trong các vụ vi phạm dữ liệu hay không
- **Thống kê**: Theo dõi số lượng người dùng truy cập

## Công nghệ

### Backend
- Node.js + Express + TypeScript
- API RESTful

### Frontend
- React + TypeScript + Vite
- Tailwind CSS

## Bảo mật

- Không lưu trữ mật khẩu người dùng
- Tất cả xử lý được thực hiện cục bộ
- Không gửi mật khẩu qua mạng
