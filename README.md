# PassCheck - Công cụ kiểm tra độ mạnh mật khẩu

Website tĩnh để **kiểm tra mật khẩu**, **phát hiện rò rỉ** và **tạo mật khẩu mạnh**.

## ✨ Tính năng
- **Kiểm tra độ mạnh**: Điểm số 0–100, phân tích độ dài, ký tự, entropy, gợi ý cải thiện.
- **Kiểm tra rò rỉ**: Mô phỏng API, kết quả tức thì, giao diện rõ ràng.
- **Tạo mật khẩu**: Dễ nhớ hoặc ngẫu nhiên, 8 chiến lược khác nhau, nút sao chép nhanh.
- **Thiết kế hiện đại**: Dark theme, responsive, hiệu ứng hover & thông báo đẹp mắt.
