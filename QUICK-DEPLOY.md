## Step 2: Deploy Frontend to Vercel ‚ñ≤

1. Go to [Vercel.com](https://vercel.com/) and login
2. Click "Add New Project" ‚Üí Import from GitHub
3. Select repository: `ntdovn/PassCheck`
4. **Important**: Set Root Directory to `/frontend`
5. Framework Preset: **Vite**
6. Build Command: `npm run build` (auto-detected)
7. Output Directory: `dist` (auto-detected)
8. Add Environment Variable:
   ```
   VITE_API_URL=https://YOUR-RAILWAY-URL/api
   ```
   ‚ö†Ô∏è Replace `YOUR-RAILWAY-URL` with the actual Railway URL from Step 1
9. Click "Deploy"

## Step 3: Add Custom Domain üåê

1. In Vercel Dashboard ‚Üí Go to your project
2. Click "Settings" ‚Üí "Domains"
3. Add domain: `passcheck.carterfill.me`
4. Vercel will show DNS records to add
5. Go to your DNS provider (Cloudflare, etc.) and add:
   ```
   Type: A
   Name: passcheck
   Value: 76.76.21.21
   
   OR
   
   Type: CNAME
   Name: passcheck
   Value: cname.vercel-dns.com
   ```
6. Wait 5-30 minutes for DNS propagation
7. Vercel will auto-issue SSL certificate

- Redeploy after changing env vars

- Must use `https://` not `http://`
